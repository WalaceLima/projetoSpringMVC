# projetoSpringMVC
Projeto para cadastro de funcionarios, com ênfase em CRUD no padrão de arquitetura MVC+ DTO

##Ferramentas Backend:
- Java 8;
- Gerenciador Maven
- IDE Eclipse;
- Servidor Web Tom Cat 9
- Banco MySql
- Spring Boot Framework 
  - Modulos
    -> MVC
    -> JDBC
-Criptografia MD5; 

##Ferramentas Frontend:
- HTML 5;
- CSS;
- JavaScript;
  -JQuery
- Bootstrap;
